:::tip[What apk should I use?]

**Universal** should work on all devices. If you want a smaller apk, use the one that matches your device's architecture.

:::

## The beta program on play store

Click on [this link](https://play.google.com/apps/testing/dev.linwood.quokka) to register for the [nightly](/community/nightly) version of the app. Please note that this version can have bugs. The play store version will automatically be updated when a new nightly version is released.
